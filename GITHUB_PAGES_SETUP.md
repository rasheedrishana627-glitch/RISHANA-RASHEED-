# GitHub Pages Deployment Guide

## Setup Instructions for GitHub Pages

Follow these steps to deploy your portfolio to GitHub Pages:

### Step 1: Enable GitHub Pages
1. Go to your repository: https://github.com/rasheedrishana627-glitch/RISHANA-RASHEED-
2. Click on **Settings** (top right)
3. Scroll down to **Pages** section (on the left sidebar)
4. Under "Source", select **Deploy from a branch**
5. Select **main** branch
6. Click **Save**

### Step 2: Wait for Deployment
- GitHub will automatically build and deploy your site
- You'll see a green checkmark when deployment is complete
- This usually takes 1-2 minutes

### Step 3: Access Your Site
Your portfolio will be live at:
```
https://rasheedrishana627-glitch.github.io/RISHANA-RASHEED-/
```

### Step 4: Custom Domain (Optional)
If you have a custom domain:
1. Go to Settings > Pages
2. Under "Custom domain", enter your domain name
3. Add a CNAME record to your domain provider pointing to: `rasheedrishana627-glitch.github.io`

## Troubleshooting

**Site not showing?**
- Wait 1-2 minutes for deployment to complete
- Check that Settings > Pages shows "Your site is live"
- Clear your browser cache and try again

**Want to make changes?**
- Edit files in the repository
- Push to main branch
- Changes will be live in ~1 minute
