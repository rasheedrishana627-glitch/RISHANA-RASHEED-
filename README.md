# Rishana Rasheed - Portfolio Website

A modern, responsive multi-page portfolio website showcasing my skills, projects, education, and contact information.

## 📁 Website Structure

```
├── index.html          # Homepage with hero section and highlights
├── about.html          # About me, education, skills, and learning journey
├── projects.html       # Showcase of projects and technical work
├── contact.html        # Contact information and contact form
├── styles.css          # Shared stylesheet for all pages
├── script.js           # JavaScript for interactivity
└── README.md           # This file
```

## 🎯 Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Glassmorphism design with smooth animations and transitions
- **Navigation**: Fixed navigation bar with active page highlighting
- **Multi-Page Structure**: Organized content across 4 main pages
- **Contact Form**: Integrated contact form for getting in touch
- **Project Showcase**: Dedicated section to display projects with descriptions
- **Mobile Menu**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Smooth scroll behavior throughout the website
- **Intersection Observer**: Elements fade in as they come into view

## 🎨 Design Highlights

- **Color Scheme**: Dark theme with blue and purple gradients
- **Typography**: Poppins font family for modern look
- **Effects**: 
  - Glassmorphism (blur and transparency effects)
  - Gradient text effects
  - Hover animations
  - Fade-in animations on scroll

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   ```

2. **Open in browser**
   - Simply open `index.html` in any modern web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

3. **Customize**
   - Update personal information in HTML files
   - Modify color scheme in `styles.css`
   - Add your projects in `projects.html`
   - Update contact details in `contact.html`

## 💻 Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Flexbox, Grid, Gradients, Animations
- **JavaScript (Vanilla)**: DOM manipulation, form handling, scroll effects
- **Responsive Design**: Mobile-first approach with media queries

## 📱 Responsive Breakpoints

- **Desktop**: 1024px and above
- **Tablet**: 768px to 1023px
- **Mobile**: Below 768px
- **Small Mobile**: Below 480px

## 🔧 Customization

### To update personal information:
1. Open respective HTML files
2. Replace placeholders with your information
3. Update contact details in `contact.html`

### To change colors:
1. Find color values in `styles.css`
2. Update gradient colors (currently blue `#3b82f6` and purple `#8b5cf6`)
3. Modify accent color (currently light blue `#60a5fa`)

### To add more projects:
1. Go to `projects.html`
2. Copy a project card template
3. Update title, description, and skills
4. Add to the projects grid
